# MSA-3-NovaMarket

Проектная работа 3-го спринта курса «Микросервисная архитектура» — проектирование архитектуры маркетплейса NovaMarket.

## Структура

- [Task1](./Task1) — Событийная архитектура (C4) и Saga-хореография оформления заказа.
- [Task2](./Task2) — История заказов: обновлённая диаграмма и ADR.
- [Task3](./Task3) — Динамическое масштабирование в Kubernetes (HPA по памяти и RPS).
- [Task4](./Task4) — NGINX: Rate Limiter и Circuit Breaker.
